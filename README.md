## Goal & Use Case

<details>
<summary>Expand to learn more</summary>

This repository is a **data-driven directory** of the Bangladeshi developer community on GitHub. We aim to highlight impactful contributors, discover rising talent, and provide a platform for developers to showcase their work. It serves as a resource for finding collaborators, mentors, and exploring the local open-source ecosystem.
</details>

## How to Join

<details>
<summary>Expand for instructions</summary>

Adding yourself is **fully automated**. Follow these steps:

### 1. Open an Issue
Use the [Add Developer](https://github.com/sharf-shawon/Awesome-Bangladeshi-Devs/issues/new?template=add_developer.yml) template.

### 2. Submit
Our automation fetches your stats and adds you to the list.

### 3. Removal
Use the [Remove Developer](https://github.com/sharf-shawon/Awesome-Bangladeshi-Devs/issues/new?template=remove_developer.yml) template for self-removal.
</details>

## How it Works

<details>
<summary>Expand for ranking details</summary>

Profiles are ranked using a **weighted composite score** based on the last 90 days of GitHub activity:

- 35% Contributions: Total activity (commits, PRs, issues, etc.).
- 20% Followers: Impact and reach.
- 10% Pull Requests: Active collaboration.
- 10% Stars: Community recognition.
- 10% Public Repos: Portfolio size.
- 15% Others: Reviews (5%), Issues (5%), and Commits (5%).
</details>
